# Finance Data Processing and Access Control Backend

A backend project for managing financial data, user roles, access control, and dashboard analytics.

This project is being built as part of a backend development assignment to demonstrate backend architecture, SQL database integration, API design, validation, business logic, and role-based access control.

## Current Status

The project setup is completed up to the database schema stage.

### Completed so far
- Project initialized with Node.js and Express
- Environment variables configured using `dotenv`
- Neon PostgreSQL database connected successfully
- Basic server setup completed
- Database health check route added
- Final backend folder structure planned
- SQL schema created for:
  - `users`
  - `financial_records`

### Not completed yet
- User authentication
- Role-based authorization middleware
- User management APIs
- Financial records CRUD APIs
- Dashboard summary APIs
- Input validation layer
- Centralized error handling
- Deployment

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Neon Database
- dotenv
- pg
