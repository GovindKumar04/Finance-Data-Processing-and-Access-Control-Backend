# Finance Data Processing and Access Control Backend

A backend system for managing financial records, user roles, access control, and dashboard-level analytics.

This project is built as part of a backend engineering assignment to demonstrate API design, SQL data modeling, business logic handling, validation, error handling, and role-based access control in a clean and maintainable way.

---

## Features

- User management
- Role-based access control
- Financial record CRUD operations
- Record filtering
